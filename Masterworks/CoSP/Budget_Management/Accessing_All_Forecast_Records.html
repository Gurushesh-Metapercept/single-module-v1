Viewing the Forecast

In the FORECAST LIST page, by default, all approved forecast records are displayed. You can view the forecast distribution of budget estimate items.
  1. In the module menu, click Projects.
    The PROJECTS list page is displayed.
  2. In the list page, double-click the required project, and in the navigation pane, expand the project folder.
  3. In the navigation pane, expand Budget Management, and then click Forecasts.
    The FORECAST LIST page is displayed.
  4. In the list page, select the appropriate forecast record, and then click View.
    The GENERAL DETAILS page is displayed.
    The GENERAL DETAILS page displays the following non-editable fields:
    Field NameDescription
    IDAn identification number of the forecast.
    DescriptionThe detailed description of the forecast.
    NotesThe notes on the forecast.
    Start DateThe start date of the forecast.
    End DateThe end date of the forecast.
    Created OnThe date on which the forecast is created.
    Created ByThe name of the user who created this forecast.
    Approved OnThe date on which the record was approved.
    Approved ByThe name of the approver.
    The FORECAST DETAILS page displays the following non-editable fields:
    Column NameDescription
    NameName of the forecast is displayed.
    Start DateThe start date of the forecast.
    End DateThe end date of the forecast.
    Duration Based on the start date and end date, the duration is displayed.
    CurveDisplays the distribution curve that is applied to the budget.
    UnitThe unit for the item as defined in the budget estimate.
    Amount in $Displays the total Amount in $ for each item from the CURRENT BUDGET ITEMS page.
    Remaining Amount in $It is calculated as: Amount in $ - Sum of Amounts in Forecast

    This helps the Project Manager understand if there is a overshoot or undershoot of the budget.

    <<Month Year>> Displays budget distribution across various months of the project.
    You can toggle between the monthly or yearly view using the Monthly View and Yearly View toolbar options, respectively.
    Note: The default cost distribution, as monthly or yearly, is configured in the Budget Management page of the Administration module. For more information, refer to Configuring Budget Management
